# Numerical Methods in Python

This project contains Python implementations of foundational numerical methods commonly used in scientific computing and engineering analysis.

## Project Objective
The objective of this project is to build reusable and well-documented Python implementations of iterative numerical techniques used to solve mathematical problems when exact analytical solutions are difficult or unavailable.

## Implemented Methods
- Newton-Raphson method for solving non-linear equations
- Root-finding techniques using iterative approximation
- Finite difference approximation for numerical derivatives

## Key Features
- Modular function-based implementations
- Convergence checks using tolerance thresholds
- Simple test examples for validation
- Clear and readable Python structure

## Technical Concepts
- Iterative Solvers
- Convergence Criteria
- Error Tolerance
- Root Approximation
- Numerical Differentiation
- Scientific Programming

## Tools and Libraries
- Python

## File Structure
- `newton_raphson.py` — Newton-Raphson implementation
- `finite_difference.py` — numerical derivative approximation
- `root_finding.py` — root-solving implementation
- `README.md` — project overview
